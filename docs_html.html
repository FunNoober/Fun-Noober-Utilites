Text Editor

"Other" Tab

  • Select All Selects All The Text In The Document
  • Copy Copies The Text In The Document
  • Paste Pastes The Text In Your Clipboard
  • Caps All Capitilizes All The Text In The Document
  • Lower All Lowercases All The Text In The Document

"Settings" Tab

  • Syntax Highlighting Turns On Basic Highlighting of Parenthesis and Quotes
  • Show Line Numbers Displays The Line Number

Code Editor

Run Command

  • command>menu_load>go Loads The Menu
  • command>kill_proccess>go Closes The Program
  • command>syntax_highlight_off>go Disables Syntax Highlighting
  • command>disable_line_numbers>go Disables The Display Of Line Numbers

Program Scheduler

Times

  • Launch Time Is The Time The Program Is Launched
  • Terminate Time Is The Time The Program Is Terminated
  • 24 Hour Clock Is The Only Time Supported

Control Buttons

  • Choose Application Opens A Window To Choose The Application To Launch and Terminate
  • Choose Null Application Chooses A Application That Does Not Exist
  • Launch Application Istantly Launches The Application
  • Terminate Application Istantly Terminates The Application
